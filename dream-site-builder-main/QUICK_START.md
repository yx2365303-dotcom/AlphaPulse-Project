## ❌ 常见错误速查

### 错误1: "用户账号不存在"

**原因：** 没执行 SQL 脚本

**解决：** 重新执行第一步

---

### 错误2: 数据显示为空

**原因：** 数据库没有数据

**解决：** 
1. 检查 Tushare 历史回填是否完成
2. 在 Supabase 控制台查询：
   ```sql
   SELECT COUNT(*) FROM limit_list_ths;
   ```
3. 如果为0，说明数据还没导入

---

### 错误3: 页面一直加载

**原因：** API 请求失败

**解决：**
1. 按 F12 打开控制台
2. 查看 Network 标签
3. 检查是否有红色请求
4. 查看错误信息
